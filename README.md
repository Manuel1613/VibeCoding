# VibeCoding

## Temática
Landing page sobre consumo energético de distintos modelos de IA según la cantidad de tokens procesados.

## Prompts usados

### PROMPT 1
Quiero que me hagas una landing page sencilla usando HTML, CSS y JavaScript (vanilla, sin frameworks).  
Debe ser responsive y mostrar gráficos comparativos del consumo energético de distintos modelos de IA por cantidad de tokens.  
Los colores y la estética deben ser futuristas y profesionales.

### PROMPT 2
Añade un selector para elegir el modelo de IA y mostrar automáticamente su consumo energético mensual en un gráfico.  

### PROMPT 3
Modifica la comparativa para mostrar también el costo económico aproximado por token.
